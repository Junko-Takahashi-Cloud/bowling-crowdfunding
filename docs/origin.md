# スポーツボウリング場構想が生まれた理由

私が初めてボウリング場を訪れたのは、小学校4年生の頃でした。子供会の行事だったと思います。

ボウリング場に足を踏み入れた瞬間、目に飛び込んできたのは、おそろいのユニフォームを着た大人たちがグループで楽しそうに投球している姿でした。その光景に、胸の奥で「いつか私も、あの人たちのように投げてみたい」と強く思ったのです。

ボウリングをしたこともない子どもが、なぜそう思ったのか、理由は分かりません。それでも、その瞬間の強い憧れだけは、何十年経った今でも鮮明に胸に残っています。

◇

それから長い年月が流れ、社会人になったある日、婚活イベントで「プロボウラーによる5回レッスン」という企画に出会いました。募集人数は30名。運よく参加できた私は、レッスンを重ねるたびに、子どもの頃に抱いた憧れが少しずつ現実になっていくのを感じていました。

5回目のレッスンが終わる頃、ボウリング場の方から「サークルを作ってみませんか？」と声を掛けていただきました。

10人で始まったそのサークルは、私にとって特別な場所となりました。小学生の頃、遠くから眺めることしかできなかった「グループでボウリングを楽しむ」という夢が、形になった瞬間でした。ユニフォームこそ揃えていませんでしたが、仲間と一緒に投げられることがただ嬉しくて、毎週ボウリング場へ通いました。

サークルは約7年間続き、その間にマイボールとマイシューズを購入し、女子プロボウラーの資格試験にも挑戦しました。

結婚を機にサークルは解散しましたが、ボウリングへの思いが消えることはありませんでした。むしろ、深く関わるほどに新しい興味が湧いてきたのです。

「レーンにはどんなオイルが塗られているんだろう」
「いろんなオイルパターンを試してみたい」

レーンの裏側を見てみたいという思いも強くなりましたが、大人がレーンの裏側を見る機会は滅多にありません。競技として上達したい、オイルパターンを研究したいと思っても、自分のペースで納得するまで練習できる環境は限られていました。

その気持ちが膨らんでいったある日、ふと思いました。

「自分の練習場があったらいいのに」

野球をする人が庭にバッティングネットを置くように。バスケットボールをする人が自宅にゴールを設置するように。私も、自分が思う存分練習できる「マイ練習場」がほしかったのです。

豪華な施設である必要はありません。2レーンあれば十分。オイルパターンを自由に試し、自分の課題と向き合える場所。

最初は、「自分のための練習場」という小さな願いでした。しかし、その思いは少しずつ変化していきます。

「もし私がそう思うなら、同じように感じているボウラーもいるのではないか」

もっと自由に練習したい人、もっとオイルパターンを研究したい人、競技としてボウリングと向き合いたい人――そんな人たちが集まり、安心して腕を磨ける場所をつくりたい。

自分ひとりの願いから始まった思いは、やがて「スポーツボウリング場構想」という夢へとつながっていきました。このプロジェクトは、私の原点を形にし、その夢を少しずつ現実へ近づけるための第一歩です。
